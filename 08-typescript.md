#### Generics

- обобщения типов - позволяют описывать множество типов от одного генерика

#### Защитники

- Специальные конструкции, которые позволяют сузить круг допустимых типов.

#### Отличия и сходства Type от Interface

- Сходства
    - Интерфейсы и типы могут использоваться для описания структур данных:
    - Могут использоваться для типизации функции:
    - Интерфейсы и типы могут быть имплементированы классами
    - Интерфейсы и типы позволяют выразить пересечение типов
- Различия
    - Интерфейсы нельзя комбинировать с mapped типами (Required, Pick, Readonly, Partial и прочими):
    - Интерфейсы позволяют выразить пересечение типов, однако не позволяют выразить объединение.
    ```js
    type Wish = | { fast: true, quality: true, cheap: false } // Дорого | { fast: true, quality: false, cheap: true } //
    Некачественно | { fast: false, quality: true, cheap: true } // Медленно

    const wish: Wish = { fast: true, quality: true, cheap: true } // Не компилируется
    ```
